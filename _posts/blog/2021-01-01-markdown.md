## Markdown이란?

마크다운(markdown)은 일반 텍스트 문서의 양식을 편집하는 문법이다. GitHub의 README 파일이나 온라인 문서, 혹은 일반 텍스트 편집기로 문서들을 편집할 때 쓰인다. 마크다운을 이용해 작성된 문서는 쉽게 HTML 등 다른 문서형태로 변환이 가능하다. 즉, 개발자들이 쉽게 문서화를 할 수 있도록 도와주는 언어이다.

